# 06. Отчеты и CSV-экспорт

## Отчеты администратора

Администратору доступны:

- отчет по объекту;
- сводка бюджета объекта;
- CSV-выгрузка.

## Отчет по объекту

Вход:

- объект.

Показатели:

- бюджет;
- сумма расходов;
- остаток;
- количество расходов;
- список последних расходов.

Строка расхода:

- дата;
- тип;
- наименование;
- сумма;
- автор;
- комментарий, если есть.

## CSV-экспорт

Доступен только администратору.

В MVP CSV формируется из карточки выбранного объекта, без пагинации и без фильтра по датам.

Колонки:

```text
Дата, Объект, Тип, Наименование, Количество, Ед.изм., Цена, Сумма, Автор, Комментарий
```

## Формат CSV

Правила:

- кодировка: UTF-8;
- разделитель: запятая;
- строки с запятыми, кавычками или переводами строк экранируются по стандартным правилам CSV;
- даты в часовом поясе сервиса;
- удаленные расходы не попадают в экспорт по умолчанию.

Предварительный формат даты:

```text
YYYY-MM-DD HH:mm
```

## Разрезы отчетов MVP

Минимальный набор разрезов:

- объект.

Фильтры по датам в MVP не входят.

Фильтры после MVP:

- дата от;
- дата до;
- тип расхода;
- поиск по наименованию.
