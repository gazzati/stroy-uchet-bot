# 01. Обзор системы

## Назначение

СтройУчет Бот - закрытый Telegram-бот для учета расходов по строительным объектам.

Система помогает:

- вести список строительных объектов;
- назначать бригадиров на объекты;
- фиксировать расходы по объектам;
- контролировать бюджет, сумму расходов и остаток;
- формировать отчеты;
- экспортировать данные в CSV.

## Пользователи

В MVP есть две роли:

- администратор;
- бригадир.

Неавторизованные пользователи не получают доступ к функциям бота.

## Границы MVP

Входит в MVP:

- авторизация по Telegram ID;
- роли пользователей;
- управление объектами;
- управление бюджетом объекта;
- управление бригадирами;
- назначение бригадиров на объекты;
- добавление расходов четырех типов;
- просмотр расходов;
- отчеты;
- CSV-экспорт;
- архивирование и восстановление объектов.

Не входит в MVP:

- OCR чеков;
- Excel-экспорт;
- веб-панель;
- подтверждение расходов;
- уведомления о превышении бюджета;
- полнотекстовый поиск;
- сложная аналитика;
- фильтры отчетов по датам;
- собственное файловое хранилище для фото документов.

## Основные ограничения

- Основной интерфейс - Telegram.
- Доступ определяется по Telegram ID.
- Первый администратор создается через переменную окружения `ADMIN_TELEGRAM_IDS`.
- Бригадиры создаются администраторами.
- Один бригадир может быть назначен на несколько объектов.
- Один объект может иметь несколько бригадиров.
- Все расходы должны относиться к конкретному объекту.
- Новые расходы по архивным объектам запрещены.
- Превышение бюджета не запрещает добавление расхода.
- Все суммы хранятся в копейках, валюта MVP - рубли.
- Фото документов в MVP хранятся через Telegram `file_id`, без скачивания в отдельное хранилище.

## Термины

| Термин | Значение |
| --- | --- |
| Объект | Строительный объект, по которому ведется учет |
| Бригадир | Пользователь, который добавляет расходы по назначенным объектам |
| Расход | Любая финансовая операция по объекту |
| Бюджет | Плановая сумма по объекту |
| Остаток | Бюджет минус сумма расходов |
| Архивный объект | Завершенный или скрытый объект, по которому нельзя добавлять расходы |
