# Спецификация проекта СтройУчет Бот

Эта папка содержит исходное ТЗ и рабочую спецификацию будущего сервиса.

## Файлы

- `StroyUchet_Bot_Full_TZ.docx` - исходное техническое задание.
- `specification.md` - нормализованная продуктовая и техническая спецификация для разработки.

## Назначение спецификации

Спецификация нужна, чтобы перевести исходное ТЗ в проверяемые требования:

- роли и права доступа;
- сущности и связи;
- бизнес-правила;
- пользовательские сценарии;
- состав MVP;
- принятые решения по спорным вопросам.
